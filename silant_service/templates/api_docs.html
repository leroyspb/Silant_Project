Доступные endpoints

📋 Машины

GET /api/machines/

Получить список всех машин

GET /api/machines/{id}/

Получить детальную информацию о машине

🔧 Техническое обслуживание (ТО)

GET /api/maintenances/

Получить список всех ТО

GET /api/maintenances/{id}/

Получить детальную информацию о ТО

⚠️ Рекламации

GET /api/complaints/

Получить список всех рекламаций

GET /api/complaints/{id}/

Получить детальную информацию о рекламации
