# Электронная сервисная книжка "Мой Силант"

## 📋 О проекте

Веб-приложение для отслеживания состояния складской техники «Силант».  
Позволяет контролировать комплектацию машин, историю технического обслуживания и рекламаций.

### Основные возможности:
- Просмотр информации о машинах, ТО и рекламациях
- Фильтрация и сортировка данных
- Разграничение прав доступа (Клиент, Сервисная организация, Менеджер)
- Добавление/редактирование/удаление записей (в зависимости от прав)
- REST API для получения данных в формате JSON
- Адаптивный дизайн под все устройства

## 🚀 Быстрый старт
```bash
git clone https://github.com/leroyspb/Silant_Project
```

### 1. Перейдите в папку проекта
```bash
cd \Silant_Project\Silant_Project
```
### 2. Активируйте виртуальное окружение

#### Windows

```bash
.\.venv\Scripts\activate   
```

#### macOS/Linux
```bash
python3 -m venv venv
source venv/bin/activate
```
Переход в папку проекта 

```bash
cd silant_service
```
### 3. Установите зависимости
```bash
pip install -r requirements.txt
```
### 4. Выполните миграции
```bash
python manage.py migrate
```
### 5. Запустите сервер
```bash
python manage.py runserver
```
### Приложение будет доступно по адресу: 
http://127.0.0.1:8000/
